# 🤖 Basic Python Chatbot

A simple command-line chatbot developed using Python and conditional (if-else) statements. The chatbot responds to user inputs with predefined answers, demonstrating the fundamentals of Python programming and conversational logic.

## 📌 Project Overview

This project was created to practice core Python concepts such as:

- Conditional Statements (if, elif, else)
- User Input Handling
- String Manipulation
- Program Flow Control
- Interactive Command-Line Applications

The chatbot can answer basic greetings and predefined questions based on user input.

## ✨ Features

- Interactive conversation through the terminal
- Predefined responses to common questions
- Simple and beginner-friendly code structure
- Easy to customize and extend
- Lightweight with no external dependencies

## 🛠️ Technologies Used

- Python 3.x
